# Kwork Tracker (Parser)

Десктопное приложение на Python (Tkinter) для автоматического мониторинга новых заказов в категории **"Разработка и IT"** на бирже фриланса Kwork.ru.

Приложение помогает оперативно реагировать на новые проекты, отправляя звуковые сигналы и всплывающие уведомления прямо на рабочий стол Windows.

## Основные возможности

- **Мониторинг в реальном времени**: Автоматический опрос ленты проектов Kwork с настраиваемым интервалом (по умолчанию каждые 5 минут).
- **Гибкие фильтры**:
  - Фильтрация по ключевым словам (через запятую).
  - Настройка диапазона цен (минимальная и максимальная стоимость заказа).
- **Уведомления**:
  - Всплывающие окна (Toast) в углу экрана со ссылкой на заказ.
  - Звуковое оповещение (системный сигнал Windows).
- **Удобный интерфейс (GUI)**:
  - Таблица со списком подходящих проектов.
  - Цветовая подсветка цен: жёлтый цвет для дорогих заказов (≥5000 руб.), голубой для средних (≥2000 руб.).
  - Просмотр подробного описания проекта по двойному клику.
  - Контекстное меню по правой кнопке мыши (открыть в браузере, скопировать ссылку, удалить из списка).
- **Дополнительные функции**:
  - Быстрый поиск и фильтрация по списку уже найденных проектов.
  - Экспорт результатов в формат `.csv`.
  - Автоматическое сохранение настроек поиска между запусками в файл `kwork_settings.json`.
  - Дедупликация (приложение запоминает отправленные проекты в файле `sent_projects.txt` и не беспокоит вас повторно).

## Установка и запуск (из исходного кода)

Для запуска приложения вам понадобится установленный Python версии 3.8 или выше.

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/ваш-логин/kwork-parser.git
   cd kwork-parser
   ```

2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите приложение**:
   ```bash
   python desktop_app.py
   ```

---

## Сборка в автономный `.exe` файл (для Windows)

Вы можете собрать приложение в один исполняемый `.exe` файл, который можно будет запускать на любом компьютере с ОС Windows без установленного Python.

1. Установите PyInstaller:
   ```bash
   pip install pyinstaller
   ```

2. Соберите проект:
   ```bash
   pyinstaller --onefile --windowed --name "KworkTracker" desktop_app.py
   ```

После успешной сборки готовый файл `KworkTracker.exe` появится в папке `dist/`.
